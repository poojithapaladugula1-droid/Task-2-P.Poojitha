# Data Analytics Project 2 - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project is part of the DecodeLabs Data Analytics Internship. It focuses on performing Exploratory Data Analysis (EDA) on a cleaned dataset to identify patterns, trends, correlations, and outliers using Python.

## 🎯 Objectives

- Analyze the cleaned dataset.
- Calculate descriptive statistics (mean, median, count).
- Detect missing values and duplicates.
- Identify outliers using box plots.
- Visualize data using charts.
- Find relationships between variables using correlation analysis.
- Summarize key insights.

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- PyCharm

## 📊 Analysis Performed

- Dataset Overview
- Data Type Analysis
- Missing Value Analysis
- Duplicate Record Check
- Descriptive Statistics
- Mean, Median, Count
- Correlation Matrix
- Heatmap
- Histograms
- Box Plots
- Count Plots
- Pair Plot
- Key Insights

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/poojithapaladugula1-droid/Task-2-P.Poojitha.git
   ```

2. Open the project in **PyCharm**.

3. Install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```

4. Place the dataset (`Cleaned_Data_Analytics_Dataset.xlsx`) in the project folder.

5. Run the Python script:
   ```bash
   python eda.py
   ```
